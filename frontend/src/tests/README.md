# Тесты для компонентов авторизации и логина

В этой директории содержатся тесты для проверки работы компонентов авторизации и логина.

## Запуск тестов

Для запуска тестов используйте команду:

```bash
npm test
```

## Структура тестов

### Backend тесты

Тесты на серверной части проверяют:
- Корректность обработки запросов на регистрацию
- Корректность обработки запросов на логин
- Проверку валидации данных
- Обработку ошибок

Файл с тестами: `backend/tests/integration/test_auth.py`

### Frontend тесты

Тесты на клиентской части проверяют:
- Отображение форм авторизации и регистрации
- Переключение между формами
- Обработку ответов от сервера
- Отображение ошибок
- Перенаправление после успешной авторизации

Файл с тестами: `frontend/src/tests/LoginForm.test.js`

## Запуск всех тестов

Для запуска backend тестов используйте:

```bash
cd backend
python -m pytest tests/integration/test_auth.py -v
```

Для запуска frontend тестов используйте:

```bash
cd frontend
npm test
```
